What is this?

InkCoverage.app calculates the ink coverage percentage for each channel in a PDF — Cyan, Magenta, Yellow, Black, and spot colors — over any selected area. Used in flexo, offset, and digital prepress to estimate ink consumption, verify print specifications, and control print costs.

How to Use InkCoverage

1

Upload a PDF

Click the upload area or drag and drop a PDF file (up to 50 MB). Your file is processed on the server and automatically deleted within 10 minutes.

2

Select a page

If your PDF has multiple pages, use the page selector in the sidebar to navigate between them. A preview of the current page will be shown.

3

Draw a crop area (optional)

Click and drag on the preview to select a specific region to analyze. Resize the crop by dragging the corner handles. If no crop is drawn, the full page will be analyzed.

4

Analyze

Click the Analyze Ink Coverage button. The results panel will show the coverage percentage for each ink channel: Cyan, Magenta, Yellow, Black, and any spot colors present in the PDF.

5

Read the results

Each ink channel shows the percentage of the selected area covered by that ink. The totals section shows combined CMYK coverage, spot color coverage, and the grand total. These values are essential for prepress quality control.

Tip: Use the crop tool to measure ink coverage on specific design elements like logos, barcodes, or color patches rather than the full page.
