Privacy Policy

Last updated: April 20, 2026

InkCoverage.app ("we", "our", "the Service") is a free, open-source online tool for analyzing ink coverage in PDF files. We are committed to protecting your privacy.

1. What We Collect

We collect as little data as possible to operate the Service:

  • Uploaded PDF files — temporarily stored on our server solely to perform ink coverage analysis. Files are automatically and permanently deleted within 10 minutes of upload.
  • IP addresses — used only for rate limiting (to prevent abuse). IP addresses are held in memory only and are not written to disk or stored permanently.
  • Basic server logs — standard web server access logs (timestamps, request paths, HTTP status codes). These do not contain file contents.

2. What We Do NOT Collect

  • We do not require user accounts or registration.
  • We do not collect names, email addresses, or any personal information.
  • We do not store or retain your PDF files beyond the 10-minute processing window.
  • We do not share any data with third parties (except as described in the Advertising section below).

3. Uploaded Files

Your uploaded PDF files are:

  • Transmitted over HTTPS (encrypted in transit).
  • Stored temporarily in an isolated directory on our server.
  • Processed only for ink coverage analysis using Ghostscript.
  • Automatically deleted within 10 minutes, regardless of whether analysis was performed.
  • Never shared with, sold to, or accessed by any third party.

4. Cookies and Analytics

We use Google Analytics (GA4) to understand how visitors use the Service. This helps us improve the tool over time. Specifically:

  • Google Analytics collects anonymized usage data such as page views, session duration, and approximate geographic location (country/city level).
  • No personally identifiable information is collected through analytics.
  • You may opt out of Google Analytics tracking by using the Google Analytics Opt-out Browser Add-on.

InkCoverage.app does not display advertisements and does not use advertising cookies. We do not set any first-party cookies.

5. Third-Party Services

  • Google Analytics — for anonymous usage analytics. Google Privacy Policy.
  • Cloudflare — for CDN, DDoS protection, and SSL. Cloudflare Privacy Policy.
  • Ko-fi — optional voluntary support (no account required to use the Service). Ko-fi Privacy Policy.

6. Data Security

We take reasonable measures to protect your data, including HTTPS encryption, server-level access controls, automatic file cleanup, and running the application in a containerized environment.

7. Children's Privacy

The Service is not directed at children under 13. We do not knowingly collect data from children.

8. Open Source

InkCoverage.app is open-source software. You can review the complete source code to verify our data handling practices on GitHub.

9. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated "Last updated" date.

10. Contact

If you have questions about this Privacy Policy, please open an issue on our GitHub repository.
